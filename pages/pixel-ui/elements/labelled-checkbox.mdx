# PIXEL.LabelledCheckbox

- **Base panel:** `Panel`
- **Source:** [cl_labelled_checkbox.lua](https://github.com/TomDotBat/pixel-ui/blob/master/lua/pixelui/elements/cl_labelled_checkbox.lua)
- **Description:** PIXEL checkbox with attached label.

## Methods

### `PIXEL.LabelledCheckbox:OnToggled(enabled)`

Callback fired when the checkbox toggles.

**Parameters**

- `enabled` (`boolean`) — Current checked state.

### `PIXEL.LabelledCheckbox:SetText(text)`

Sets the label text.

**Parameters**

- `text` (`string`) — Label text.

### `PIXEL.LabelledCheckbox:GetText()`

Gets the label text.

**Returns**

- (`string`) `text` — Current label text.

### `PIXEL.LabelledCheckbox:SetFont(font)`

Sets the label font.

**Parameters**

- `font` (`string`) — Font name.

### `PIXEL.LabelledCheckbox:GetFont()`

Gets the label font.

**Returns**

- (`string`) `font` — Current font name.

### `PIXEL.LabelledCheckbox:SetTextColor(col)`

Sets the label text color.

**Parameters**

- `col` (`Color`) — Text color.

### `PIXEL.LabelledCheckbox:GetTextColor()`

Gets the label text color.

**Returns**

- (`Color`) `col` — Current text color.

### `PIXEL.LabelledCheckbox:SetAutoWrap(enabled)`

Enables or disables automatic label wrapping.

**Parameters**

- `enabled` (`boolean`) — Whether wrapping is enabled.

### `PIXEL.LabelledCheckbox:GetAutoWrap()`

Returns whether automatic label wrapping is enabled.

**Returns**

- (`boolean`) `enabled` — Whether wrapping is enabled.
