# PIXEL.Navbar

- **Base panel:** `Panel`
- **Source:** [cl_navbar.lua](https://github.com/TomDotBat/pixel-ui/blob/master/lua/pixelui/elements/cl_navbar.lua)
- **Description:** PIXEL navbar container panel.

## Methods

### `PIXEL.Navbar:AddItem(id, name, doClick, order, color)`

Adds a selectable item to the navbar.

**Parameters**

- `id` (`any`) — Unique item identifier.
- `name` (`string`) — Item label.
- `doClick` (`fun(item:Panel)`) — Callback run when selected.
- `order` (`number|nil`) — Sort order for the item.
- `color` (`Color|nil`) — Selection color for the item.

### `PIXEL.Navbar:RemoveItem(id)`

Removes an item from the navbar by identifier.

**Parameters**

- `id` (`any`) — Item identifier.

### `PIXEL.Navbar:SelectItem(id)`

Selects an existing navbar item.

**Parameters**

- `id` (`any`) — Item identifier.
