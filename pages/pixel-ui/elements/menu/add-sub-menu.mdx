# AddSubMenu

`PIXEL.Menu:AddSubMenu(strText, funcFunction)`

Creates a submenu option and returns the submenu panel.

**Parameters**

- `strText` (`string`) — Option label.
- `funcFunction` (`fun()|nil`) — Optional click callback.

**Returns**

- (`PIXEL.Menu`) `subMenu` — Created submenu.
- (`PIXEL.MenuOption`) `option` — Parent option panel.
