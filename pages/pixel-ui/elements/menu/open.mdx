# Open

`PIXEL.Menu:Open(x, y, skipanimation, ownerpanel)`

Opens the menu at screen coordinates.

**Parameters**

- `x` (`number|nil`) — Screen X coordinate.
- `y` (`number|nil`) — Screen Y coordinate.
- `skipanimation` (`boolean|nil`) — Unused legacy parameter.
- `ownerpanel` (`Panel|nil`) — Optional panel the menu is attached to.
