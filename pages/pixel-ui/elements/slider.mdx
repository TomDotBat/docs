# PIXEL.Slider

- **Base panel:** `PIXEL.Button`
- **Source:** [cl_slider.lua](https://github.com/TomDotBat/pixel-ui/blob/master/lua/pixelui/elements/cl_slider.lua)
- **Description:** PIXEL horizontal slider control.

## Methods

### `PIXEL.Slider:OnValueChanged(fraction)`

Callback fired when slider value changes.

**Parameters**

- `fraction` (`number`) — Current normalized value in the range [0, 1].
