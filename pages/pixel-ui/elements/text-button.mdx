# PIXEL.TextButton

- **Base panel:** `PIXEL.Button`
- **Source:** [cl_text_button.lua](https://github.com/TomDotBat/pixel-ui/blob/master/lua/pixelui/elements/cl_text_button.lua)
- **Description:** PIXEL button with styled text label.

## Methods

### `PIXEL.TextButton:SizeToText()`

Resizes the button to fit its current text.

### `PIXEL.TextButton:PaintExtra(w, h)`

Draws the button text over the base PIXEL.Button background.

**Parameters**

- `w` (`number`) — Panel width.
- `h` (`number`) — Panel height.
