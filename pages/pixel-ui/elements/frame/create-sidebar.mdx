# CreateSidebar

`PIXEL.Frame:CreateSidebar(defaultItem, imageURL, imageScale, imageYOffset, buttonYOffset)`

Creates a sidebar attached to the frame.

**Parameters**

- `defaultItem` (`any|nil`) — Optional item ID to auto-select.
- `imageURL` (`string|nil`) — Sidebar logo URL or Imgur ID.
- `imageScale` (`number|nil`) — Sidebar logo scale.
- `imageYOffset` (`number|nil`) — Sidebar logo Y offset.
- `buttonYOffset` (`number|nil`) — Extra top offset for buttons.

**Returns**

- (`PIXEL.Sidebar|nil`) `sidebar` — Created sidebar panel.
