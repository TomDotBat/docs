# SizeThink

`PIXEL.Frame:SizeThink(targetPanel, hoverPanel)`

Handles resize cursor state and resizing movement.

**Parameters**

- `targetPanel` (`Panel`) — Target frame being resized.
- `hoverPanel` (`Panel|nil`) — Optional hovered panel override.

**Returns**

- (`boolean|nil`) `isSizing` — True when resize handling is active.
