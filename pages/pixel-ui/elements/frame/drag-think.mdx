# DragThink

`PIXEL.Frame:DragThink(targetPanel, hoverPanel)`

Handles drag cursor state and dragging movement.

**Parameters**

- `targetPanel` (`Panel`) — Target frame being dragged.
- `hoverPanel` (`Panel|nil`) — Optional hovered panel override.

**Returns**

- (`boolean|nil`) `isDragging` — True when drag handling is active.
