# PIXEL.Label

- **Base panel:** `Panel`
- **Source:** [cl_label.lua](https://github.com/TomDotBat/pixel-ui/blob/master/lua/pixelui/elements/cl_label.lua)
- **Description:** PIXEL text label panel.

## Methods

### `PIXEL.Label:SetText(text)`

Sets label text and stores original text for wrapping.

**Parameters**

- `text` (`string`) — Label text.

### `PIXEL.Label:CalculateSize()`

Calculates text dimensions using the current font.

**Returns**

- (`number`) `width` — Text width.
- (`number`) `height` — Text height.
