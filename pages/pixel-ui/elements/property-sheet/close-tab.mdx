# CloseTab

`PIXEL.PropertySheet:CloseTab(tab, removePanelToo)`

Closes a tab and optionally removes its content panel.

**Parameters**

- `tab` (`PIXEL.Tab`) — Tab to close.
- `removePanelToo` (`boolean|nil`) — When true, removes the content panel.

**Returns**

- (`Panel|nil`) `panel` — Detached content panel when preserved.
