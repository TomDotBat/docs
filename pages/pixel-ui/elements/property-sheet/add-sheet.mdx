# AddSheet

`PIXEL.PropertySheet:AddSheet(label, panel, material, noStretchX, noStretchY, tooltip)`

Adds a new tab and content panel to the property sheet.

**Parameters**

- `label` (`string`) — Tab label.
- `panel` (`Panel`) — Content panel.
- `material` (`any|nil`) — Unused legacy icon argument.
- `noStretchX` (`boolean|nil`) — When true, panel width is not stretched.
- `noStretchY` (`boolean|nil`) — When true, panel height is not stretched.
- `tooltip` (`string|nil`) — Tab tooltip text.

**Returns**

- (`table|nil`) `sheet` — Created sheet descriptor.
