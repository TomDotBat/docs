# PIXEL.PropertySheet

- **Base panel:** `Panel`
- **Source:** [cl_property_sheet.lua](https://github.com/TomDotBat/pixel-ui/blob/master/lua/pixelui/elements/cl_property_sheet.lua)
- **Description:** PIXEL property sheet panel with tab strip.

## Methods

### `PIXEL.PropertySheet:AddSheet(label, panel, material, noStretchX, noStretchY, tooltip)`

Adds a new tab and content panel to the property sheet.

**Parameters**

- `label` (`string`) ‚Äî Tab label.
- `panel` (`Panel`) ‚Äî Content panel.
- `material` (`any|nil`) ‚Äî Unused legacy icon argument.
- `noStretchX` (`boolean|nil`) ‚Äî When true, panel width is not stretched.
- `noStretchY` (`boolean|nil`) ‚Äî When true, panel height is not stretched.
- `tooltip` (`string|nil`) ‚Äî Tab tooltip text.

**Returns**

- (`table|nil`) `sheet` ‚Äî Created sheet descriptor.

### `PIXEL.PropertySheet:SetActiveTab(active)`

Sets the currently active tab.

**Parameters**

- `active` (`PIXEL.Tab`) ‚Äî Tab to activate.

### `PIXEL.PropertySheet:OnActiveTabChanged(old, new)`

Callback fired when active tab changes.

**Parameters**

- `old` (`PIXEL.Tab`) ‚Äî Previously active tab.
- `new` (`PIXEL.Tab`) ‚Äî Newly active tab.

### `PIXEL.PropertySheet:GetItems()`

Returns all sheet entries.

**Returns**

- (`table`) `items` ‚Äî Registered sheet descriptors.

### `PIXEL.PropertySheet:SizeToContentWidth()`

Resizes the sheet to fit the widest content panel.

### `PIXEL.PropertySheet:SwitchToName(name)`

Switches to the first tab matching a name.

**Parameters**

- `name` (`string`) ‚Äî Tab name to activate.

**Returns**

- (`boolean`) `found` ‚Äî True when a tab was found and activated.

### `PIXEL.PropertySheet:CloseTab(tab, removePanelToo)`

Closes a tab and optionally removes its content panel.

**Parameters**

- `tab` (`PIXEL.Tab`) ‚Äî Tab to close.
- `removePanelToo` (`boolean|nil`) ‚Äî When true, removes the content panel.

**Returns**

- (`Panel|nil`) `panel` ‚Äî Detached content panel when preserved.
