# PIXEL.Avatar

- **Base panel:** `Panel`
- **Source:** [cl_avatar.lua](https://github.com/TomDotBat/pixel-ui/blob/master/lua/pixelui/elements/cl_avatar.lua)
- **Description:** PIXEL circular avatar panel.

## Methods

### `PIXEL.Avatar:SetPlayer(ply, size)`

Sets the player shown by the avatar panel.

**Parameters**

- `ply` (`Player`) — Player entity to render.
- `size` (`number|nil`) — Avatar image resolution hint.

### `PIXEL.Avatar:SetSteamID(id, size)`

Sets the SteamID used by the avatar panel.

**Parameters**

- `id` (`string`) — SteamID64 or SteamID string.
- `size` (`number|nil`) — Avatar image resolution hint.
