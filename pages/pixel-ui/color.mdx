## DecToHex

```lua
PIXEL.DecToHex(dec, zeros?)
```

Converts a decimal value to a hexadecimal string representation.

**Parameters:**

1. **dec**: number - The decimal value to convert.
2. **zeros**: _number_ - The number of leading zeros to add to the hexadecimal string. Defaults to 2.

**Returns**

1. **hex**: string - The hexadecimal string representation of the decimal value.

## ColorToHSL

```lua
PIXEL.ColorToHSL(color)
```

Converts a color to its HSL (Hue, Saturation, Lightness) representation.

**Parameters:**

1. **color**: `Color` - The color to convert.

**Returns:**

1. **h**: number - The hue value of the color.
2. **s**: number - The saturation value of the color.
3. **l**: number - The lightness value of the color.

## HSLToColor

```lua
PIXEL.HSLToColor(h, s, l, a)
```

Converts HSL (Hue, Saturation, Lightness) values to a color.

**Parameters:**

1. **h**: number - The hue value.
2. **s**: number - The saturation value.
3. **l**: number - The lightness value.
4. **a**: _number_ - The alpha (opacity) value. Defaults to 1.

**Returns:**

1. **color**: `Color` - The color created from the HSL values.

## OffsetColor

```lua

```

Offsets the RGB values of a color by adding the specified offset.

**Parameters:**

1. **color**: `Color` - The color to offset.
2. **offset**: number - The offset value to add to each RGB component of the color.

**Returns:**

1. **offsetColor**: `Color` - A new color object with the offset RGB values.

## LerpColor

```lua
PIXEL.LerpColor(t, from, to)
```

Linearly interpolates between two colors.

**Parameters:**

1. **t**: number - The interpolation value between 0 and 1.
2. **from** `Color` - The starting color.
3. **to**: `Color` - The ending color.

**Returns:**

1. **lerpedColor**: `Color` - The interpolated color between from and to.
