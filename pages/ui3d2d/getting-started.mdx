# Getting Started
## Installation
Including UI3D2D in your project is a simple process, it can be downloaded form the GitHub releases page [here](https://github.com/TomDotBat/ui3d2d/releases/latest).
Once downloaded, the source files can be copied into your project and included using the [include](https://wiki.facepunch.com/gmod/Global.include) method.

<Callout type="info" emoji="ℹ️">
	The UI3D2D extras are not required for the main library to function and can be removed from your project if not necessary.
</Callout>
